0.1 Create a Project

RStudio projects make it easy to mange our work, by putting everything in our a working directory, the .proj extention maintains our workspace, history, and source files. RStudio projects are associated with a directory on your computer. I recomend creating a directory on your computer to put all of the class material in there.

To create a Project Select: File > New Project

0.2 Use R Notebooks!

I plain hate R script(hate is a strong word, really just dislike), and fortunatly there is an alternative - *** R Notebooks*** . Notebooks are a cool, modern interface that allows you to document your code and results all in an interactive development environment. An R Notebook is an “R Markdown” document with chunks-o-code that can be executed independently and interactively, with output visible immediately beneath the input.
