Welcome

This is the course website for “WakeForest’s MSBA Technical Bootcamp in R”. The goal of this five day course is to help you get comforatable with using R and RStudio for data science. You’ll learn how to get your data into R, shape it, transform it, and visualise it. After working through each day’s exercises, you’ll have the tools to tackle a wide variety of data science challgens using R.

What is R?

When you see powerful analytics, statistics, machine learning and visualizations used by data scientists, chances are that the R language is behind them. Open-source R is the analtyics programming language that data experts the world over use for everything from mapping broad social and marketing trends online to developing financial and climate models that help drive our economies and communities. R is one of the most popular programing languages for data science and statistics. R is popular because:

  1. R is open source i.e. it’s free
  2. R has latest ML methods
  3. R is easy to learn
  4. R is fun

Setup R & RStudio

You need to install two pieces of software. First you download and install R, then you download and install RStudio. RStudio is the interactive development enviornment that we’ll be using. In fact this website was created entirely with RStudio.

  1. Download and install R, you can get it for free at https://cran.rstudio.com/

  2. Download and install RStudio, you can get that free at RStudio

  3. Once you’ve installed R & RStudio check that it’s working by

Selecting: File > New File > R Script

Keep things up-to-date

A new major version of R comes out once a year, and there are 2-3 minor releases each year. It’s a good idea to update regularly. Upgrading can be a bit of a hassle, especially for major versions, which require you to reinstall all your packages, but putting it off only makes it worse.
